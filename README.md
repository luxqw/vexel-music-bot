## ðŸ“¦ Quick Start (Docker)

### 1. Clone the repository

```bash
git clone -b https://github.com/luxqw/vexel-music-bot.git
cd vexel-music-bot
```

### 2. Create .env

You can use `.env.example` as a template.

### 3. Start with Docker Compose

```bash
docker compose up -d
```
